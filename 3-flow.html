How to enumerate items?

  1. First
  2. Second
  3. Third
  4. Fourth
  5. Fifth

Example: chemical elements

>>> stuff = ['hydrogen', 'helium', 'lithium']

Classic way

>>> stuff = ['hydrogen', 'helium', 'lithium']
>>> for i in range(len(stuff)):
...     print(i + 1, stuff[i])
...
1 hydrogen
2 helium
3 lithium

Use zip and range for numbering

>>> for number, name in zip(range(1, len(stuff) + 1), stuff):
...     print(number, name)
...
1 hydrogen
2 helium
3 lithium

enumerate

>>> for number, name in enumerate(stuff):
...     print(number, name)
...
0 hydrogen
1 helium
2 lithium

enumerate

>>> for number, name in enumerate(stuff, start=1):
...     print(number, name)
...
1 hydrogen
2 helium
3 lithium
